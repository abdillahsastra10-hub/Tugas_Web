# Penghitung Nilai Yudisium

Repository ini berisi Laporan sebuah Web sederhana yang dirancang untuk menghitung nilai yudisium berdasarkan input nilai praktek dan teori. Aplikasi ini memungkinkan pengguna untuk menghitung rata-rata nilai dan mendapatkan hasil predikat yudisium secara otomatis, yang dikategorikan dalam beberapa level, seperti 'Sangat Memuaskan', 'Memuaskan', 'Cukup Memuaskan', dan lainnya.

---

## Fitur Utama:

1. Input Nilai: Pengguna dapat memasukkan nilai praktek dan teori yang digunakan untuk perhitungan yudisium.
2. Perhitungan Nilai: Sistem menghitung rata-rata dari nilai praktek dan teori untuk menentukan nilai akhir.
3. Predikat Yudisium: Berdasarkan nilai rata-rata yang dihitung, aplikasi memberikan predikat yudisium, yaitu:
* A (Sangat Memuaskan
* B (Memuaskan)
* C (Cukup Memuaskan)
* D (Kurang Memuaskan)
4. Desain Responsif: Antarmuka aplikasi ini menggunakan font Poppins dan memiliki desain yang ramah pengguna.

---

## Teknologi yang Digunakan:
HTML, CSS, JavaScript.

---

## Cara Penggunaan:

* Masukkan nilai praktek pada kolom yang tersedia.
* Masukkan nilai teori pada kolom yang tersedia.
* Klik tombol "Hitung Yudisium" untuk melihat hasilnya.
